## Testownik
Multiple choice quiz which get data from txt on web server.

Google play : https://play.google.com/store/apps/details?id=com.vango.testownik&hl=pl

### Technologies, features 

- Java
- Recycler View 
- Shared Preferences
- AsyncTask
- Reading txt file from web ```BufferedReader reader = new BufferedReader(new InputStreamReader((new URL(urlAdress)).openStream(),"UTF-8"));```
- Save data from app to remote database using php
